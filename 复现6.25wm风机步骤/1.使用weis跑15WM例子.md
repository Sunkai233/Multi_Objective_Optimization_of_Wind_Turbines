# 使用weis跑例子生成openfast与输出文件详讲

## weis下载

### 创建 conda 环境

```
conda config --add channels conda-forge
conda env create --name weis-env -f environment.yml
conda activate weis-env
```

## 安装必要包

Windows 下 `petsc4py` 一般不装，只需要安装 `mpi4py`：

```
conda install mpi4py
```

然后安装 WEIS 本体：

```
pip install -e .
```

## 生成文件：

### OpenFAST 模型文件

- `.fst` - 主输入文件
- `.dat` - 各子模块数据文件（AeroDyn、ServoDyn、ElastoDyn等）
- `.bld` - 叶片数据文件
- `.twr` - 塔架数据文件

### ROSCO 控制器文件

- `DISCON.IN` - 控制器输入参数
- `Cp_Ct_Cq.txt` - 性能系数表
- `.yaml` - 控制器配置文件

### 仿真输出

- `.out` - OpenFAST 时域仿真结果
- `.sum` - 仿真摘要文件
- 载荷统计文件

#### 1. 塔架建模差异
