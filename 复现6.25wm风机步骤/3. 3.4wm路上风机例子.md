## 配置文件使用说明

此配置文件是OpenFAST/WISDEM等风电仿真软件的标准输入格式，包含了：

1. **几何定义**：叶片、轮毂、机舱、塔架的详细几何参数
2. **材料属性**：复合材料、金属材料的力学性能和成本
3. **气动数据**：翼型坐标和极曲线数据
4. **控制策略**：变桨和转矩控制参数
5. **环境条件**：风况、土壤等环境参数
6. **成本模型**：详细的部件和运营成本

通过修改这些参数，可以进行风机设计优化、载荷分析、经济性评估等工程分析。

## 载荷工况说明 (DLC Descriptions)

### IEC 61400-1 标准载荷工况分类:

| DLC  | 描述                | 目的         |
| ---- | ------------------- | ------------ |
| 1.1  | 正常湍流模型 (NTM)  | 疲劳载荷分析 |
| 1.2  | 正常湍流模型 (50年) | 极限载荷分析 |
| 1.3  | 极端湍流模型 (ETM)  | 极限载荷分析 |
| 1.4  | 极端相干阵风 (ECG)  | 极限载荷分析 |
| 1.5  | 极端风向变化 (EWS)  | 极限载荷分析 |
| 5.1  | 紧急停机            | 故障载荷分析 |
| 6.1  | 停机 + 极端风速     | 极限载荷分析 |
| 6.3  | 停机 + 偏航误差     | 极限载荷分析 |
| 6.4  | 空转工况            | 特殊载荷分析 |

### 重要参数说明:

- **analysis_time**: 实际工程中通常为600秒，这里设为1秒用于快速测试
- **transient_time**: 瞬态时间，通常设为30-60秒以消除初始瞬态效应
- **n_seeds**: 随机种子数，工程中通常需要6个以上保证统计可靠性
- **ws_bin_size**: 风速分组间隔，影响载荷谱的精度

## 配置文件使用建议

1. **开发阶段**: 使用较短的仿真时间和较少的种子数进行快速验证
2. **验证阶段**: 逐步增加仿真时间和种子数
3. **认证阶段**: 使用完整的IEC标准参数进行最终分析

此配置支持从快速概念设计到详细载荷认证的全流程分析。
